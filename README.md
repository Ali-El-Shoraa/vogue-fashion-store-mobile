**Vogue Fashion Store** is a modern e-commerce mobile application built with React Native. It offers a seamless shopping experience for high-end fashion enthusiasts, right from their mobile devices.

---

## ✨ Key Features

- **Trendy Product Browsing:** Explore the latest fashion trends with an elegant UI.
- **Cart Management:** Easily add or remove items from your shopping cart.
- **Material 3 Design:** Responsive interface supporting dynamic themes.
- **Automated Builds:** APK is automatically updated via GitHub Actions.

---

## 🚀 Installation & Download

You can download the latest APK from the **Releases** section on GitHub.

> [!TIP]
> You can also find and install this app directly through the **RepoStore** app by searching for **Vogue Fashion Store**.

---

## ⚙️ Tech Stack

- **Framework:** React Native (Expo)
- **State Management:** Context API
- **Backend:** Node.js (Local / Supabase soon)
- **CI/CD:** GitHub Actions
